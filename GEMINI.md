# 프로젝트 컨텍스트: timer-app

## 개요
자격증 평가용 타이머 웹 애플리케이션입니다. 사용자가 시험 시간을 설정하고, 특정 간격(5~30분)으로 알림을 받을 수 있습니다.

## 기술 스택
- **프레임워크:** React (Vite)
- **언어:** JavaScript
- **스타일링:** CSS Modules 또는 표준 CSS

## 기능 목록
1.  **타이머 설정:** 전체 시험 시간 입력.
2.  **카운트다운:** 남은 시간 표시.
3.  **알림 설정:** 5~30분 단위 반복 알림 (Web Audio API 비프음).
4.  **시각적 알림:** '나노바나나' 로고 및 알림 메시지.

## 실행 방법
1.  의존성 설치 (최초 1회):
    ```bash
    npm install
    ```
2.  개발 서버 실행:
    ```bash
    npm run dev
    ```
3.  브라우저에서 터미널에 표시된 주소(보통 `http://localhost:5173`)로 접속합니다.

## 개발 규칙
- 모든 주석 및 문서는 한국어로 작성합니다.
- 코드는 직관적이고 유지보수 가능하게 작성합니다.
